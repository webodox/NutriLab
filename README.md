# How to compile and run NutriLab:
**Prerequisites:** Java JDK, Android SDK, Pixel 5 Device Emulator, or Android Studio

**Steps:**
1. Clone the project repository from GitHub here.
2. Open terminal and go to the project root directory.
3. Run ./gradlew assembleDebug (Mac) or ./gradlew.bat assembleDebug (Windows) to compile the project.
4. Start an Android emulator, preferably Pixel 5.
5. Run ./gradlew installDebug to install the app onto the device
6. The app will launch on the device.
